# Guide

Welcome to the REMQ guide! Here you'll find everything you need to get started with Redis Enhanced Message Queue.

## What is REMQ?

REMQ (Redis Enhanced Message Queue) is a high-performance message queue system for Deno. Built on Redis, it offers reliable task scheduling and processing with Deno's modern runtime.
