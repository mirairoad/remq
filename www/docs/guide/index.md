# Guide

Welcome to the REMQ guide! Here you'll find everything you need to get started with Redis Enhanced Message Queue.

## What is REMQ?

REMQ (Redis Enhanced Message Queue) is a Redis Streams-based job queue and worker system for Deno. Built on Redis, it offers reliable task scheduling and processing with Deno's modern runtime.

The guide focuses on concepts and workflows. For full API options and types,
refer to the [API Reference](/reference/).
